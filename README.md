### RPN Calculator
Built with Java 11.  
Library used:
* Findbugs
* Junit 5  

Build system:
* gradle

#### Run, test

* clone the git repository:  
    `git clone https://github.com/mberionni/rpncalculator.git`
* `cd rpncalculator`

* `gradle build`  
* `gradle test`, if you only want to run unit tests.
* `cd build/classes/java/main/`
* start the command-line application:  
    `java com.RPNCalc`
